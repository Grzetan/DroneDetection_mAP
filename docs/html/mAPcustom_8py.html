Calculates mean average precision for every value in given IOU range.

For iou_start=0.5, iou_stop=0.9, iou_step=0.1 these IOU thresholds will be checked: [0.5, 0.6, 0.7, 0.8, 0.9].

Parameters
predictionsNumpy array of detected bboxes. Bbox format: [frame_id, x1, y1, x2, y2, score]: ndarray
labelsNumpy array of ground truth bboxes. Bbox format: [frame_id, x1, y1, x2, y2]: ndarray
iou_startStarting IOU threshold.
iou_stopEnding IOU threshold.
iou_stepStep of thresholds. Total number of checked thresholds depend on this number.
plotIf true precision vs recall graph is shown which can visualize area under curve
Returns
Returns mean average precision score for given IOU range: float <0, 1>
